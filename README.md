This depository contains files pertaining to sample & practical certification exams from DataCamp. More specifically, they directly pertain to exams done in Python, R, &/or SQL.

All four of the SQL projects were in March, 2025. The Python Data Associate Certifications (sample & practical) were both done in November, 2024. The Data Analyst projects in Python (sample & practical) were done in October, 2025 & November, 2025 respectively.
